# Kashf - AI-powered Electricity Consumption Assistant

An 
AI-powered electricity management platform that helps Egyptian households monitor electricity consumption, avoid costly Sheriha (billing tier) jumps, and receive personalized energy-saving recommendations through smart meter and bill scanning. Features dedicated user and admin dashboards for consumption tracking, analytics, and management, and can be installed on mobile devices and desktops as a Progressive Web App (PWA) for a seamless app-like experience.
